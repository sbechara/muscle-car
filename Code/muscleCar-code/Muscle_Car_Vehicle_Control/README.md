# Vehicle Control Hookup Guide
Make the following connections for the car.

| Arduino Pin | RF Reciever |
|:-----------:|:-----------:|
| GND         | Pin 1       |
| GND         | Pin 6       |
| GND         | Pin 7       |
| 5V          | Pin 4       |
| 5V          | Pin 5       |
| Digital 7   | Pin 2       |

**Note: Pin 3 on RF reciever is not supposed to be connected to anything**
